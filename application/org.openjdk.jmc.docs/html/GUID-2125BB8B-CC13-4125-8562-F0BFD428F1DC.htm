Adding a Custom JVM Connection

It is possible to add connections to favorite JVMs so that they are always available in the JVM Browser.

To add a new JVM connection:

  1. Click the Create a new custom JVM connection button on top of the JVM Browser.
    Alternatively, open the File menu, select Connect, then select Create a new connection and click Next.
  2. In the New Connection Wizard, enter the host name and port number to connect to, and specify the connection name.
  3. If you want to test the connection and verify that you have entered the correct values, click Test Connection.
  4. If necessary, specify the user name and password for authentication, and select Store credentials in settings file to encrypt them using a master password. For more information, see Storing Credentials in Settings File.
  5. Click Finish to end the custom connection setup or Next to select a tool to connect to the JVM.

    The wizard tests the connection. If the test is successful, the next screen appears.

  6. (Optional) If you clicked Next at the previous step, choose one of the various listed actions for the new connection, such as Start JMX Console or Start Flight Recording.
  7. Click Finish.
