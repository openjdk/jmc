Executing Operations

Operations are mechanisms that cause the execution of code on the JMX server within the managed object.

Operations can take parameters and return a result, like methods. JDK Mission Control supports the execution of these operations, along with passing parameters and displaying the results.

The Operations subtab in the MBean Browser lists the operations associated with the selected MBean and allows you to execute any of those operations. The return type is displayed next to the name of the operation in the list. The table to the right contains a list of parameters required to execute the operation, and allows you to set the values for each parameter.

To execute an operation:

  1. Select the operation you want to execute.
  2. Provide values for all parameters required by the application.
  3. Click Execute under the table. The result of the operation is displayed in a separate panel with tabs for each executed operation.
