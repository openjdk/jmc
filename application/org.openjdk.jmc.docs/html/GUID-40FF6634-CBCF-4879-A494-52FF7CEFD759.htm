Viewing Recordings

The Flight Recorder plugin enables you to view JFR recording files.

The JFR runtime engine dumps recorded data to files with the jfr extension. These JFR recordings are binary files for viewing in the JDK Mission Control (JMC) client.

Note:

Java SE Development Kit (JDK) 7 update 4 or later version is required to view JFR recordings.

The recording file is automatically opened in the JMC client when a timed recording finishes or when a dump of a running recording is created. You can also open any recording file by double-clicking it or by opening the File menu, selecting Open File and locating the necessary recording file in the file system explorer.

This topic describes general procedures for viewing the recordings in the JMC client. For information about how the data in various tabs is organized, see Flight Recorder Tab Reference.
