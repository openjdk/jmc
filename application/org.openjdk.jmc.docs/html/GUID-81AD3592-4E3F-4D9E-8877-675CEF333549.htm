Using the Environment Variables Page

The Environment Variables page contains details about the environment variables for the system on which the recording was made.

Information on this page is presented in a table, with search capability.

Using the Environment Variables Page: Tables

  • Key: The name of the environment variable.

  • Value: The value assigned to the environment variable.
