Using and Customizing the JVM Browser

The JVM Browser contains a list of available JVM connections. You can expand and right-click the node with the connection to show available services, actions and settings. JVM connections support drag-and-drop operations.

To customize the JVM Browser settings:
  1. Open the Window menu and select Preferences.
  2. In the left pane, expand JDK Mission Control and select JVM Browser.
  3. Enter the node highlight time in milliseconds that defines how long a node with a newly discovered JVM connection is highlighted in the JVM Browser tree.
  4. Select Confirm deletion of nodes to enable the confirmation prompt for deleting a JVM connection from the JVM Browser tree.
  5. Select Warn when no local JVMs are detected if such a warning is desired.
  6. Click Apply to apply changes.
