Dashboard

The Dashboard panel consists of dials (or performance meters) that display the current value of a single attribute.

The dial is a colored scale with values and a pointer. Above the dial is the title. In the middle of each dial, the current units are displayed. Below the dial are the current value of the attribute (that corresponds to the current position of the pointer) and the maximum value (that corresponds to the watermark, which is the highest observed position of the pointer). The title, units, and color of both the scale and the watermark can be adjusted.

By default, the following dials are displayed:

  • Used Java Heap Memory

  • JVM CPU Usage

  • Live Set + Fragmentation
