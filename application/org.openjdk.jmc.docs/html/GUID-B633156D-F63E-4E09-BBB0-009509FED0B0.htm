2 Using the JDK Mission Control Client

The JMC client is the main application from which you connect to JVMs and start the tools included in JDK Mission Control.

You can run the JMC client either as a standalone application or as an Eclipse plug-in. Ensure that you run the JMC client in a secure environment, because it does not include or run with a security manager.
