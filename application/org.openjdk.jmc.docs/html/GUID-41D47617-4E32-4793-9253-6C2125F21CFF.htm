Customizing JMX Console Settings

Use the following steps to change the preferences of the JMX Console:

  1. Open the Window menu and select Preferences.
  2. In the left pane, expand JDK Mission Control and select JMX Console.
  3. Select Show warning before updating heap histogram if necessary.
  4. Change the options in the subsections:

    • Communication Settings

    • Diagnostic Commands Preferences

    • MBean Browser Preferences

    • JMX Data Persistence Settings

  5. Click Apply to apply changes.
