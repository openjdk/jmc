Communication Settings

The following options control how the JMX Console communicates with the JVM.

  • Default update interval [ms]: Defines the time interval that the JMX Console uses to update the attributes.

  • Retained event values: The max amount of values to allow a single subscription (for example a subscribed attribute, such as machine CPU load) to use.

  • Mail server (SMTP): Defines the address of the mail server that is used when sending notifications.

  • Mail server port: Defines the port number to be used for the specified mail server.

  • Secure mail server (SSL): Specifies whether the communication through SMTP should use SSL authentication.

  • Mail server user: Defines the e-mail address from which notifications are sent.

  • Mail server password: Defines the password for the specified e-mail address.
