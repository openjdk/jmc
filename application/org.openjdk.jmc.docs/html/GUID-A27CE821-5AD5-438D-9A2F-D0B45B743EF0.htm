Using the Recording Page

The Recordings page contains information about the recordings and their event settings.

This page displays its information in tables and graphs.

Using the Recording Page: Tables and Graphs

Concurrent Recordings Table

  • Recording Name: The name of the recording.

  • Id: The ID of the recording.

  • Recording Start: The recording start time.

  • Recording Duration: The recording duration.

Event Settings Graphs

  • Setting For: Indicates what this setting is for.

  • Setting Name: The setting name.

  • Setting Value: The setting value.

Using the Recording Page: Configuring Rules

You can set configuration attributes for the rules associated with this page by clicking the Edit Configuration icon.

The following options are available:

Lost Flight Recorder Buffers

  • Buffer lost limit: The number of lost buffer events needed to trigger a warning.
