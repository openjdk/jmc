Adding or Editing a Dial

It is possible to add a dial to the Dashboard panel.

To add a dial, click the Add attributes button in the top right corner of the panel and select the attribute in the Select Attributes to Add window.

To edit a dial’s properties:

  1. Right-click the dial and select Edit Display Name to set the dial’s name.
  2. Right-click the dial and select Properties to set values for the dial’s gradient and watermark. Specify the gradient start and end values to adjust the colors of the dial. You can also adjust the colors by clicking the Colors button and selecting the start and end colors, as well as the color of the watermark.
  3. Select whether you want to show the watermark.
  4. Click OK.

To move a dial within the Dashboard panel, right-click it and select either Move left or Move right.

To remove a dial, right-click it and select Delete.

To reset the watermark to the current value of the attribute, right-click the dial and select Clear Statistics. This is useful when you have been monitoring the application for a long time and believe that the performance has changed so that the maximum for an attribute should now be different.
