6 Using the Flight Recorder Plugin

The Flight Recorder plugin integrates the functionality of the JDK Flight Recorder (JFR) runtime engine into JDK Mission Control. The intuitive graphical user interface (GUI) enables you to start, stop, configure and view recordings created using JFR.

Note:

JFR is not enabled in the JVM by default in versions prior to JDK 11. For these versions, it is available only in commercial packages based on Java Platform, Standard Edition (Oracle Java SE Advanced and Oracle Java SE Suite). For information about enabling JFR, see Enabling Flight Recorder.

This topic contains the following sections:

  • Starting a Flight Recording describes how to work with the Start Flight Recording Wizard.

  • Dumping a Running Flight Recording describes how to dump the current data in the global buffer to a recording file.

  • Managing Flight Recording Templates describes how to work with the Flight Recorder Templates Manager.

  • Configuring Flight Recorder Settings describes how to change the settings of the Flight Recorder plugin.

For information about viewing the recordings using the Flight Recorder plugin, see: Viewing Recordings.
