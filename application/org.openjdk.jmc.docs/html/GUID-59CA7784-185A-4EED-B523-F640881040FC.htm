Working With Graphs

The JMX Console uses graphs to show how attributes change their values in time.

To the right of every graph is a legend of all attributes added to the graph. You can use the check boxes next to the attributes to display or hide them in the graph. For more information about adding and removing attributes for a graph, see Working with Attributes.

To switch a graph to table format that is accessible to users of screen readers, click the Accessibility mode button in the top right corner of the graph. To enable accessibility mode for all graphs by default:

  1. Open the Window menu and select Preferences.
  2. In the left pane, select JDK Mission Control.
  3. Under Accessibility Options, select Use accessibility mode.
  4. Click Apply to apply changes.
Toggle the Enable updates button to start/stop the updating of any given graph.

To modify the appearance of a graph, right-click it and use the following context menu items:

  • Show: Select the time interval to display. You can choose between the last 15 seconds, minute, 10 minutes, hour, day, or week.

  • Y-axis Range: Select the range of the vertical axis scale. You can choose to auto-adjust the range according to the minimum and maximum displayed values, auto-adjust it only according to the maximum value and show zero in any case, or specify the minimum and maximum values manually.

  • Label Density: Select the label density on the vertical axis scale. You can choose Variable, Very dense, Dense, Normal, Sparse, and Very sparse.

  • Rendering Mode: Select between subsampling and averaging for rendering the graph.

  • Edit Titles: Set the Graph title, Y-axis title, or X-axis title of the graph.

  • Zoom In: Zoom in to a section of the graph.

  • Zoom Out: Zoom out of a section of the graph.

  • Export as Image: Save the current state of the graph as a PNG image file.
