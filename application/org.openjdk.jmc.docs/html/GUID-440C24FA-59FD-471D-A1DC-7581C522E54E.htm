Using the Overview Tab

The Overview tab is opened by default when you start the JMX Console. It displays a set of dials and graphs with real-time usage statistics for any attributes of your choice.

By default, the Overview tab contains the collapsed JMX Data Persistence Settings panel, the Dashboard panel and two graphs named Processor and Memory. The default layout provides an overview of the CPU and memory usage. You can modify the set of dials and graphs to provide you with an overview of the current state for attribute values that are most important to you.

To reset the Overview tab to the default state, click Reset to default controls in the top right corner of the tab.
