Using JMX Persistence Data

The JMX Persistence Data plugin combines graphs of attributes for which data persistence was enabled in the Overview tab of the JMX Console. JMX Persistence Data is a low-overhead way to monitor attribute trends in a single tab.

To enable JMX Persistence Data logging, expand the JMX Persistence Data Settings panel at the top of the Overview tab and click the Activate JMX Persistence Data persistence button in the top right corner. Only the attributes in the list are logged. Logging starts when you enable it and stops when you disable it or disconnect the JVM. Unlike graphs in the JMX Console, where only the buffered attribute values are displayed, the JMX Persistence Data log is permanently saved to disk, so it is possible to display as much JMX Persistence Data in the graphs as the log files allow. For more information, see Using JMX Data Persistence Files.

To open the JMX Persistence Data tab, in the JVM Browser, expand the node with the JVM for which you want to view JMX persistence data, right click the MBean Server node and select Open JMX Persistence Data.

To add a graph to the JMX Persistence Data tab, click the Add Chart button in the top right corner of the tab. To add attributes to a graph, click the Add attributes button in the top right corner of the graph. To remove an attribute from the graph, right-click it in the graph legend and select Delete.

By default, all data is shown for each attribute added to the graph. You can select the attributes that you want to show or hide on the graph using the check boxes in the graph legend.

There are several ways to manipulate the time range of the displayed data in the graph:

  • You can zoom into a specific range on the graph by clicking and dragging with the mouse.

  • You can right-click the graph and select Zoom In or Zoom Out.

  • You can right-click the graph, select Show and then select how far back in time you would like the graph to display (15 seconds, 1 minute, 10 minutes, 1 hour, 1 day, or 1 week).

By right-clicking the graph you can also adjust the y-axis range and label density, select the rendering mode, edit titles, and export the graph as an image.
