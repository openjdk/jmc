Using the Environment Page

The Environment page contains basic information about the environment in which the recording was made.

This page displays information in three sections: CPU, Memory, and Operating System.

CPU

  • CPU Type: The CPU type.

  • Number of Cores: The number of cores.

  • Number of Hardware Threads: The number of hardware threads.

  • Number of Sockets: The number of sockets.

  • CPU Description: A description of the CPU.

Memory

  • Available Physical Memory: The available amount of physical memory.

Operating System

  • OS Version: The operating system version.
