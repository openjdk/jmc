JVMs Discovered Over the Java Discovery Protocol

The Java Discovery Protocol (JDP) enables the JVM Browser to list JVM instances on the network.

To set JDP preferences:
  1. Open the Window menu and select Preferences.
  2. In the left pane, expand JDK Mission Control, then JVM Browser, and select JDP.
  3. Enter an address for the multicast group to join.
  4. Enter a port to listen to.
  5. Enter a timeout for the JVM heart beat.
  6. Click Apply to apply the changes.

Note:

You must restart JDK Mission Control for the changes to take effect.
