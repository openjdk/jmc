Configuring Flight Recorder Settings

It is possible to configure the settings for the Flight Recorder plugin using the JDK Mission Control preferences window.

To modify the Flight Recorder settings:
  1. Open the Window menu and select Preferences.
  2. In the left panel, expand JDK Mission Control and select Flight Recorder.
  3. Click Restore Defaults to reset the user interface settings of the Flight Recorder plugin to the default state.
  4. Select Remove finished recordings from server if you do not want to keep the finished recordings on the server
  5. Select Ask for confirmation before removing flight recording templates if you would like to confirm the deletion of a template.
  6. Select the default time span for the dump used when double-clicking a running recording.
    You can choose from the following options:

    • No default: Brings up a window asking you to specify the time span.

    • Whole recording: Dumps all the collected data.

    • Last part of recording: Allows you to specify the duration of the recording you would like to dump in the Time span to dump (last part) field below. After the value add s to specify seconds, min to specify minutes, h to specify hours, d to specify days, or w to specify weeks.

  7. Click Apply to apply the changes.
