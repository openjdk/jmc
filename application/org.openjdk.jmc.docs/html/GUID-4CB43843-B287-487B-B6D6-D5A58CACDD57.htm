Viewing Notifications

Notifications provide information about events on the server. Not all MBeans provide such information, and you have to subscribe to the notifications (if they are available) using the Notifications subtab in the MBean Browser.

To subscribe to notifications, select an MBean and then select Subscribe if it is available in the Information panel. Below, in the Log panel, each notification is added as a separate entry to the table with the name composed of the date and time of the notification. Expand the entries to view the details of the notifications.
