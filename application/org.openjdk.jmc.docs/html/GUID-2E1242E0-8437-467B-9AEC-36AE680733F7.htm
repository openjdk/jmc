Locally Discovered JVMs

The JVM Browser lists locally discovered JVMs that are running on your computer.

You can connect to these JVMs using the JDK Mission Control plug-ins even if JMXRMI was not configured for them. If you cannot connect to a locally discovered JVM, it probably means that the version of the JVM does not support JMXRMI by default, or it must be enabled manually (for example, a 64–bit version of JDK Mission Control cannot enable JMXRMI in a 32–bit JVM).
