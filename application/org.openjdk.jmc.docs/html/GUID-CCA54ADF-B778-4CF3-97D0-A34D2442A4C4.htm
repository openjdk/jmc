Graphs

By default, the Overview tab contains two graphs: Processor and Memory. The Processor graph shows CPU usage by the computer and the JVM. The Memory graph shows how system memory and the Java heap is used by the JVM.

To add a graph to the Overview tab, click the Add Chart button in the top right of the tab, then add the necessary attributes to the graph and modify its appearance as needed.

To remove a graph panel, click the Close button in the top right corner of the graph panel.

For information about adding and removing attributes for a graph, see Working with Attributes. For information about using graphs, see Working with Graphs.
