JMX Data Persistence Settings

The JMX Data Persistence Settings panel is collapsed at the top of the Overview tab in the JMX Console. It contains a list of attributes that can be saved persistently.

To add an attribute to the list, click the Add attributes button with the plus sign in the top right corner of the JMX Data Persistence Settings panel and select the attribute in the Select Attributes to Add window. For more information, see: Working with Attributes.

To remove an attribute from the list, right-click it and select Delete.

To enable data persistence for the attributes in the list, click the Activate JMX Data Persistence button in the top right corner of the Historical Data Settings panel.
