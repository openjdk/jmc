JMXコンソールは、実行中のJVMインスタンスのモニタリングと管理のためのツールです。このツールによって、メモリーとCPU使用状況、ガベージ・コレクション、スレッド・アクティビティなどのライブ・データが提供されます。また、JVMおよびJavaアプリケーションのMBeanのモニターおよび管理に使用できる完全な機能を持つJMX MBeanブラウザも含まれています。
Java Flight Recorderは、HotSpot JVMと実行中のアプリケーションに関する詳細な記録を生成します。フライト・レコーダ・ツールを使用して、記録されたプロファイルをオフラインで分析できます。記録されたデータには、メソッド、ロックおよび割当てのプロファイリング、ガベージ・コレクションの統計、最適化の決定、イベントの待機時間などが含まれています。
JDK Mission Controlは、Javaアプリケーションの管理、モニタリング、プロファイリングおよびトラブルシューティングを行うためのツール・スイートです。初回インストール時に、JDK Mission ControlはJMXコンソールとJava Flight Recorderで構成されます。Mission Controlから他のプラグインを簡単にインストールできます。
Mission Controlの機能を拡張したプラグインが使用可能です。これらはすべてMission Controlからインストールできます。「ヘルプ」→新規ソフトウェアのインストール・メニューに移動し、インストールするプラグインを選択します。使用可能なプラグインの例をいくつか示します。
JDK Mission Controlクライアントは、JVMに接続して、JDK Mission Controlに含まれるツールを開始できるアプリケーションです。
JavaアプリケーションとJVMの実行時のモニターおよび管理を行います。
JVMの実行時の動作を記録し、Java Flight Recorderツールを使用してオフラインで記録の分析を行います。
JDK Mission Controlの機能を拡張するために様々な特殊プラグインをダウンロードしてインストールします。

MBeanの参照とトリガーの作成
MBeanブラウザのMBeanを通じてJVMおよびJavaアプリケーションをモニターおよび管理し、MBean値が変更された場合にアラートをトリガーするルールを定義します。
ランタイム情報の表示
システムのプロパティ、メモリー管理、Javaスレッドなどに関する詳細なランタイム情報を取得します。
自動分析
自動分析の結果をチェックして、問題を確認し、問題を修正するためのヒントを得ます。
カスタム・ページ
カスタム・ページを作成し、必要に応じて表示を最適化します。
詳細の表示
メモリー使用状況、ガベージ・コレクション、頻繁に使用されるメソッドとクラス、スレッド、Javaロックなどに関する詳細な情報を表示します。
Weblogic Server Tab Pack for JFR
Weblogicのカスタム設計タブを使用してWeblogic Serverからフライト記録のコンテンツをより効率的に分析します。新規タブ・グループがフライト・レコーダに追加され、これによりデータベース問合せ、EJBコール、サーブレット呼出しなどを参照できます。
