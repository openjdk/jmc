Eclipseプラグインについて

JDK Mission ControlのEclipseプラグイン・エディションは、アプリケーションのプロファイリングとモニタリングを行うJDK Mission Controlのツールセットと、Eclipseの開発プラットフォームのシームレスな統合を提供します。JDK Mission ControlクライアントをEclipseと統合すると、JDK Mission Controlを構成するツールとEclipseの機能を一体化できます。JDK Mission ControlのEclipseプラグインは、JDK Mission ControlクライアントとEclipseを統合することで有効になる特別な機能の使用手順を提供します。

Eclipse IDE内でJDK Mission Controlクライアントを実行すると、IDEの機能にアクセスできます。スタンドアロンのリッチ・クライアント・プラットフォーム(RCP)アプリケーションとして実行した場合には、IDEの機能をツールで使用できません。これらのうち最も重要なのは、実行中のアプリケーションのソース・コードをJDK Mission Controlクライアントで直接開いて表示する、Jump-to-Sourceという機能です。

JDK Mission ControlクライアントのEclipse IDEとの統合におけるもう1つの利点は、開発フェーズでも本番フェーズと同様にアプリケーションのプロファイリングとモニターが行えることです。これによって、アプリケーションを本番環境に実際にデプロイする前に、起こりうるランタイムの問題を識別できます。
