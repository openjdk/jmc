使用 JMX 数据持久性文件

启用 JMX 数据持久性日志记录时，数据以二进制文件形式存储在持久性目录中。

默认情况下，JMC 7.0 的这个目录位于 USER-HOME/.jmc/7.0.0/persisted_jmx_data 中

要更改持久性目录，请执行以下操作：

  1. 打开窗口菜单，然后选择首选项。
  2. 在左侧窗格中，展开 JDK Mission Control，再展开 JMX 控制台，然后选择 JMX 数据持久性。
  3. 修改持久性目录字段中的路径，或者单击浏览在文件系统资源管理器中选择目录。
  4. 单击应用以应用更改。
持久性目录包含的文件夹标有启用了数据持久性的 JVM 连接的名称。各 JVM 连接文件夹包含的子文件夹标有启用了数据持久性的属性的名称。这些子文件夹包含名为 *.persisted_jmx_data 的实际日志文件，以及名为 series.info 的其他元数据文件。

您可以将日志文件拖放到 JMC 控制台窗口，以在 JMX 数据持久性窗口中查看这些日志文件。如果要在网络上共享数据文件，请确保针对各 JVM 连接保留了相应的文件夹结构。
