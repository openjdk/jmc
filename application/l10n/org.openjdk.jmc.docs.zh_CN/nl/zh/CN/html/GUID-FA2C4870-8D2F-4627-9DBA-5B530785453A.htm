使用 JMX 持久性数据

JMX 持久性数据插件在 JMX 控制台的概览选项卡中组合了启用了数据持久性的属性所组成的图形。JMX 持久性数据是一种用来在单个选项卡中监视属性趋势的低开销方法。

要启用 JMX 持久性数据日志记录，请展开概览选项卡顶部的 JMX 持久性数据设置面板，然后单击右上角的激活 JMX 持久性数据持久性按钮。在日志中仅记录列表中的属性。日志记录在您启用它时启动，在您禁用它或与 JVM 断开连接时停止。JMX 持久性数据日志与 JMX 控制台中的图形不同，前者永久保存到磁盘中，后者只显示缓冲的属性值，因此图形中显示的 JMX 持久性数据可以与日志文件所允许的数据一样多。有关详细信息，请参阅使用 JMX 数据持久性文件。

要打开 JMX 持久性数据选项卡，请在 JVM 浏览器中展开包含您要查看其 JMX 持久性数据的 JVM 的节点，右键单击 MBean 服务器节点并选择打开 JMX 持久性数据。

要向 JMX 持久性数据选项卡添加图形，请单击选项卡右上角的添加图表按钮。要向图形添加属性，请单击图形右上角的添加属性按钮。要从图形中删除属性，请在图形图例中右键单击该属性，然后选择删除。

默认情况下，显示添加到图形中的各属性的所有数据。您可以使用图形图例中的复选框选择要在图形上显示或隐藏的属性。

控制图形中所显示数据的时间范围有多种方法：

  • 可以通过用鼠标单击和拖动在图形上缩放到特定范围。

  • 您可以右键单击图形，然后选择放大或缩小。

  • 您可以右键单击图形，选择“显示”，然后选择希望图形在显示时间上往回拉多远（15 分钟、1 分钟、10 分钟、1 小时、1 天或 1 周）。

通过右键单击图形，您还可以调整 y 轴范围和标签密度，选择呈现模式，编辑标题以及将图形导出为图像。
