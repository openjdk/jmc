使用“进程”页

进程页包含有关记录期间运行的进程的信息。

此页以表和图形形式显示进程数据。

使用“进程”页：选择和方面

使用页顶部的下拉菜单选择选项和方面（如果需要）。通过拖动图表区域（或选择表中的数据）并从上下文菜单中选择存储选择或存储选择并设置为焦点选择，可以将其他页中的数据保存到选择下拉菜单。

下面的控件提供其他行为：

  • 显示并发：显示对于所选方面并发的所有事件。

  • 交集：显示与原始事件有交集的所有事件，或者仅显示包含在同一时间范围内的事件。

  • 所有线程：显示所有线程中的事件，或者仅显示与原始事件相关的线程中的事件。

使用“进程”页：表和图形

此图显示在一段时间内绘制的 CPU 占用率和并行进程的相关信息。

表中各列的标签如下：

  • 进程标识符：进程标识符 (PID)。

  • 命令行：启动此进程的命令。

  • 第一次采样：第一次对此进程采样的时间。

  • 上次采样：上次对此进程采样的时间。

使用“进程”页：配置规则

通过单击编辑配置图标，可以设置与此页关联的规则的配置属性。

以下选项可供使用：

争用 CPU 占用率

  • 争用 CPU 占用率限制：触发信息通知时其他进程需使用的 CPU 量。

  • 争用 CPU 占用率警告限制：触发警告时其他进程需使用的 CPU 量。

争用进程

  • 争用进程限制：触发信息通知需要同时运行的进程数。
