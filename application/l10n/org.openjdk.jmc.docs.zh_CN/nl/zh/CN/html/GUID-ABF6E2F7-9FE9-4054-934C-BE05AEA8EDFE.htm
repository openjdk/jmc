使用“文件 I/O”页

文件 I/O 页包含有关在文件系统中执行的读取/写入活动的信息。

此页以表和图形形式显示其信息。

使用“文件 I/O”页：选择和方面

使用页顶部的下拉菜单选择选项和方面（如果需要）。通过拖动图表区域（或选择表中的数据）并从上下文菜单中选择存储选择或存储选择并设置为焦点选择，可以将其他页中的数据保存到选择下拉菜单。

下面的控件提供其他行为：

  • 显示并发：显示对于所选方面并发的所有事件。

  • 交集：显示与原始事件有交集的所有事件，或者仅显示包含在同一时间范围内的事件。

  • 所有线程：显示所有线程中的事件，或者仅显示与原始事件相关的线程中的事件。

  • 时间范围：缩放到页上的相关位置的时间范围，或者清除时间范围缩放。

使用“文件 I/O”页：表和图形

表中各列的标签如下：

  • 路径：读取/写入数据的文件系统路径。

  • I/O 总时间：线程忙于 I/O 的估计总时间。实际总时间可能更长。

  • 计数：所有读取/写入计数的总和。

  • 读取计数：文件读取总数。

  • 写入计数：文件写入总数。

  • 读取的字节数：从文件系统读取的总字节数。

  • 写入的字节数：写入到文件系统的总字节数。

此页还包含三个子选项卡：

  • 时间表：显示 I/O 事件范围以及每个时段中读取/写入的数据量。

  • 持续时间：显示 I/O 事件持续时间的分布。

  • 事件日志：根据在其他表中的选择显示 I/O 事件的日志，并显示所有事件或子集。

使用“文件 I/O”页：配置规则

通过单击编辑配置图标，可以设置与此页关联的规则的配置属性。

以下选项可供使用：

文件读取峰值持续时间

  • 文件读取持续时间警告限制：触发警告所需的最短文件读取持续时间。

文件写入峰值持续时间

  • 文件写入持续时间警告限制：触发警告所需的最短文件写入持续时间。
