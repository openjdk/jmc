通过 Java Discovery Protocol 搜索到的 JVM

Java Discovery Protocol (JDP) 可使 JVM 浏览器列出网络上的 JVM 实例。

要设置 JDP 首选项，请执行以下操作：
  1. 打开窗口菜单，然后选择首选项。
  2. 在左侧窗格中，展开 JDK Mission Control，再展开 JVM 浏览器，然后选择 JDP。
  3. 输入要联接的多点传送组的地址。
  4. 输入要监听的端口。
  5. 输入 JVM 脉动超时。
  6. 单击应用以应用更改。

注：

必须重新启动 JDK Mission Control 才能使更改生效。
