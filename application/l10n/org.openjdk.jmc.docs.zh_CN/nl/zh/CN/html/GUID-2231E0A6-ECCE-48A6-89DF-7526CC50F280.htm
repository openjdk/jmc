JDK Mission Control 辅助功能模式

JDK Mission Control 显示性能数据刻度盘和图形，并提供了十分有用的图形化数据视图，可揭示趋势并帮助标识性能度量的最小值和最大值。

不过，刻度盘和图形无法以可供屏幕阅读器读取的方式传达信息。为了补救此问题，JDK Mission Control 提供了以表格式查看刻度盘和图形数据的方法，这一方法使得视觉障碍的用户更容易访问。要切换到辅助功能模式，请在带有刻度盘或图形的相关面板中，单击右上角的辅助功能模式按钮。 

要为 JDK Mission Control 中的所有面板启用辅助功能模式，请执行以下操作：

  1. 打开窗口菜单，然后选择首选项。

  2. 在左侧窗格中选择 JDK Mission Control。

  3. 在辅助功能选项下，选择使用辅助功能模式。

  4. 单击应用。
