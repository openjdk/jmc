使用“记录”页

记录页包含有关记录及其事件设置的信息。

此页以表和图形形式显示其信息。

使用“记录”页：表和图形

“并行记录”表

  • 记录名：记录名称。

  • ID：记录的 ID。

  • 记录开始时间：记录开始时间。

  • 记录持续时间：记录持续时间。

事件设置图形

  • 设置属于：指示此设置所属的对象。

  • 设置名称：设置名称。

  • 设置值：设置值。

使用“记录”页：配置规则

通过单击编辑配置图标，可以设置与此页关联的规则的配置属性。

可以使用以下选项：

丢失的飞行记录器缓冲区

  • 缓冲区丢失限制：触发警告所需的丢失缓冲区事件数。
