6 使用飞行记录器插件

飞行记录器插件将 Java 飞行记录器 (JFR) 运行时引擎的功能集成到 JDK Mission Control。通过直观的图形用户界面 (GUI)，您可以启动、停止、配置和查看使用 JFR 创建的记录。

本主题包含以下各部分：

  • “启动飞行记录”介绍如何使用启动飞行记录向导。

  • “转储正在运行的飞行记录”介绍如何将全局缓冲区中的当前数据转储到记录文件。

  • “管理飞行记录模板”介绍如何使用飞行记录器模板管理器。

  • “配置飞行记录器设置”介绍如何更改飞行记录器插件的设置。

有关如何使用飞行记录器插件查看记录的信息，请参阅“查看记录”。
