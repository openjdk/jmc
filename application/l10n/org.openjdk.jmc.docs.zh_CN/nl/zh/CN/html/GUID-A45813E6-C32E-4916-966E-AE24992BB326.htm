使用“JVM 内部”页

JVM 内部页包含有关记录期间运行的 JVM 的信息。

此页包含 JVM 信息和 JVM 标记的面板。

使用“JVM 内部”页：JVM 信息面板

  • JVM 启动时间：启动 JVM 时的日期和时间。

  • JVM 名称：JVM 的名称。

  • JVM 版本：JVM 的完整版本。

  • JVM 参数：包含用于启动 JVM 的 java 命令选项。

  • JVM 应用程序参数：包含用于运行应用程序的类文件或 JAR 文件以及传递给 main() 方法的参数。

使用“JVM 内部”页：VM 标记

JVM 标记表包含以下各列：

  • 名称：标记的名称。

  • 值：标记的值。

  • 源：标记源，默认、人机工程学、命令行或管理之一。默认情况下会设置默认选项。命令行选项在命令行中指定。人机工程学选项根据环境由 JVM 自身通过人机工程学方式设置。
