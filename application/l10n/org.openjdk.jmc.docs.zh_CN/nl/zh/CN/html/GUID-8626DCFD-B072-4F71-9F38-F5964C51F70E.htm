JDK Mission Control 客户机 GUI

首次启动 JMC 客户机时，默认情况下将打开多个视图，如下所示。

可以打开窗口菜单，依次选择显示视图和其他以打开列出所有可用视图的窗口。      选择所需视图，然后单击确定以打开它们。  可以使用以下视图：

  • JVM 浏览器：显示所有可用的 JVM 连接（默认情况下可见）。

  • 大纲：以树中页的形式组织和显示飞行记录数据以便于导航（默认情况下可见）。

  • 进度查看：显示正在运行的操作的进度，例如，飞行记录。

  • 属性：列出表中选定项的属性，包括表中未显示的隐藏属性。

  • 结果：显示规则及其对应得分的列表。

  • 堆栈跟踪：显示所记录事件的堆栈跟踪。
