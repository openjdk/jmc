添加或编辑刻度盘

可以将刻度盘添加到面板面板中。

要添加刻度盘，请单击该面板右上角的添加属性按钮，然后在选择要添加的属性窗口中选择属性。

要编辑刻度盘的属性，请执行以下操作：

  1. 右键单击刻度盘并选择编辑显示名称以设置刻度盘名称。
  2. 右键单击刻度盘并选择属性，以便设置刻度盘的渐变和水印的值。指定渐变开始和结束值来调整刻度盘上的颜色。还可以通过单击颜色按钮并选择开始和结束颜色以及水印的颜色来调整颜色。
  3. 选择是否要显示水印。
  4. 单击确定。

要移动面板面板中的刻度盘，请右键单击该刻度盘并选择左移或右移。

要删除刻度盘，请右键单击该刻度盘，然后选择删除。

要将水印重置为属性的当前值，请右键单击刻度盘，并选择清除统计信息。上述操作在以下情况非常有用：您长时间监视该应用程序，认为性能已经发生变化，因此属性的最大值现在应该不同。
