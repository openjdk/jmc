7 实验性插件

适用于 JDK Mission Control 的实验性插件是一系列旨在以多种方法扩展 JDK Mission Control 或帮助开发此类扩展的插件。

实验性插件可以是下列项之一：

  • 以后可以根据反馈将技术前瞻合并到主分发中。

  • 使用未与 Java SE 直接关联的功能扩展 JDK Mission Control 的插件，例如，适用于 Java 飞行记录器的 WebLogic Server 插件。

注：

目前不支持这些插件，这些插件仅用于评估。JDK Mission Control 产品分发不包括实验性插件。

通过使用 JDK Mission Control Client，可以从实验性更新站点下载和安装实验性插件。

要安装插件，请执行以下操作：

  1. 打开“帮助”菜单并选择“安装新软件”。

  2. 选择要安装的插件，然后按照安装向导操作。

注：

要使用已安装的插件，请重新启动 JDK Mission Control。
