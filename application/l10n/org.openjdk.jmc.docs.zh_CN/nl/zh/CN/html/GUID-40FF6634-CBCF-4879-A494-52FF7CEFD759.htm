查看记录

使用飞行记录器插件可以查看 JFR 记录文件。

JFR 运行时引擎将记录的数据转储到扩展名为 jfr 的文件中。这些 JFR 记录是二进制文件，可在 JDK Mission Control (JMC) 客户机中查看。

注：

查看 JFR 记录需要 Java SE 开发工具包 (JDK) 7 Update 4 或更高版本。

在定时记录完成时或者创建了正在运行的记录的转储时，记录文件将自动在 JMC 客户机中打开。您可以双击以打开任意记录文件，或者在文件菜单中选择打开文件并在文件系统资源管理器中定位所需的记录文件来打开该文件。

此主题介绍了用于在 JMC 客户机中查看记录的一般过程。有关数据如何在不同选项卡中排列的信息，请参阅“飞行记录器选项卡参考”。
