4 关于 JVM 浏览器

JVM 浏览器是显示可用 Java 虚拟机 (JVM) 的视图，您可以使用 JDK Mission Control (JMC) 对这些 JVM 进行监视、管理和故障排除。

JVM 浏览器在您启动 JMC 客户机时自动启动。要将插件连接到 JVM，请右键单击浏览器中的 JVM，然后从上下文菜单中选择要使用的工具。

可以自动搜索或手动配置浏览器中的 JVM。手动配置定制 JVM 对精细控制 JMXServiceURL 或设置身份证明很有用。

自动搜索到的 JVM 有以下两种类型：

  • 本地搜索到的 JVM 与 JDK Mission Control 在同一计算机上运行

  • 网络上的通过 Java Discovery Protocol 搜索到的 JVM (JDP)
