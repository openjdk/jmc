使用“诊断命令”选项卡

使用诊断命令可监视 Java 应用程序的效率和性能。JDK Mission Control 使用大量不同的诊断工具，包括一组可以使用诊断命令选项卡针对应用程序运行的命令。

诊断命令选项卡分为以下几个部分：

  • 左侧的操作列表包含命令列表，您可以对连接的 JVM 上运行的应用程序执行这些命令。

  • 右侧的命令参数列表包含您可以指定的参数的名称和值。

  • 要执行的包含所有参数的完整命令显示在底部，旁边是执行按钮（用于运行命令）和帮助按钮（用于显示命令帮助）。

  • 执行命令或请求命令帮助的结果显示在底部延伸出来的输出面板。此面板包含一个单独的选项卡，供您每次执行命令或请求命令帮助时使用。在选项卡的标题中，随完整的命令显示执行的日期和时间。

要运行命令，请在操作列表中选择它，并在右侧的表格中提供参数值，然后单击底部的执行。
