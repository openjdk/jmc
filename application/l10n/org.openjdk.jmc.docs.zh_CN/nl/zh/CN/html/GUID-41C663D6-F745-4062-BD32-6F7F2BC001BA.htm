转储正在运行的飞行记录

您可以将全局缓冲区的当前内容手动转储到记录文件。当您碰巧正在运行记录时，此功能对分析最近问题非常有用。

要转储正在运行的飞行记录，请执行以下操作：

  1. 在 JVM 浏览器中展开所需连接，然后展开飞行记录器节点。
  2. 右键单击您感兴趣的记录，然后选择转储。
  3. 通过单击浏览按钮选择记录的路径和文件名。
  4. 选择要转储事件数据的时段。可以使用以下选项：

    • 整个记录：为缓冲区中所有记录的事件创建转储。

    • 记录的最后部分：创建在记录结束前的指定时间内发生的事件的转储。

    • 记录间隔：创建在指定时间间隔内发生的事件的转储。

  5. 当查看记录文件时，如果要将所选时段用于范围导航器，请选择在双击以打开记录时使用此时间范围
  6. 单击完成以创建记录转储文件。
