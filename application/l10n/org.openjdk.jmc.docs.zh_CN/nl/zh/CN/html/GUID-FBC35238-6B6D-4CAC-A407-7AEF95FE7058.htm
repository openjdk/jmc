调整联机帮助文本的大小

有视觉障碍的用户可能会发现，如果不增大文本大小，在独立 (RCP) 版本的 JDK Mission Control 中阅读联机帮助文档会非常困难。由于 Eclipse 帮助查看器的限制，目前您无法调整其文本的大小。要更改字体大小，请在外部浏览器中查看帮助。

  1. 打开窗口菜单，然后选择首选项。
  2. 在左侧窗格中选择帮助。
  3. 在打开模式下，选择在外部浏览器中打开帮助内容。
  4. 单击应用。
JDK Mission Control 将使用在操作系统中指定的默认 Web 浏览器，您不能从 JDK Mission Control 中另外指定一个 Web 浏览器。在操作系统中指定浏览器的方式取决于操作系统。
