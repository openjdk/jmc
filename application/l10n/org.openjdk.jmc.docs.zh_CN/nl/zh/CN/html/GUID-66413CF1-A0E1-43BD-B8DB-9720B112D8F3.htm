使用“内存”页

内存页包含有关一段时间内的内存分配、使用和垃圾收集的数据。

此页以表和图形形式显示信息。

使用“内存”页：选择和方面

使用页顶部的下拉菜单选择选项和方面（如果需要）。通过拖动图表区域（或选择表中的数据）并从上下文菜单中选择存储选择或存储选择并设置为焦点选择，可以将其他页中的数据保存到选择下拉菜单。

下面的控件提供其他行为：

  • 显示并发：显示对于所选方面并发的所有事件。

  • 交集：显示与原始事件有交集的所有事件，或者仅显示包含在同一时间范围内的事件。

  • 所有线程：显示所有线程中的事件，或者仅显示与原始事件相关的线程中的事件。

  • 时间范围：缩放到页上的相关位置的时间范围，或者清除时间范围缩放。

使用“内存”页：表和图形

表中各列的标签如下：

  • 类：全限定类名。

  • 最大实时计数：垃圾收集之后实例的最大估计数量。

  • 最大实时大小：垃圾收集之后所有实例的最大估计大小。

  • 实时大小增长：第一次垃圾收集之后与最后一次垃圾收集之后的总计大小之差。

  • 总分配量：估计的总分配量（非确切值）。

您可以选择以下任一项以便在图形上显示：

  • 垃圾收集：垃圾收集导致的暂停。

  • 总分配量：估计的总分配量（非确切值）。

  • 已用堆：堆上按对象分配的内存。

  • 堆空间：已提交大小：虚拟空间的已提交内存大小。

  • 堆空间：保留大小：虚拟空间的保留内存大小。

  • 已用大小：正在使用的物理内存总量。

  • 总大小：操作系统可使用的物理内存总量。

  • 实时大小：垃圾收集之后的所有实例大小。
