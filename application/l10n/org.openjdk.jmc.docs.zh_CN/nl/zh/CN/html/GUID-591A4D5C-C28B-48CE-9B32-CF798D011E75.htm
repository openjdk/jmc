面板

“编辑器”选项卡由面板组成，其中包括逻辑相关的信息，这些信息采用图形、表、列表等格式。可以使用面板中的不同控件修改部分参数，例如按钮、文本字段、下拉列表等。

某些面板可以折叠以便为选项卡中的其他面板提供更多空间（使用黑色三角形显示和隐藏面板名称旁边的按钮）。在面板名称的右侧有按钮可用于添加和删除组件、刷新数据、切换模式、更改设置、获取有关面板的联机帮助等。

大部分面板是唯一的，但在不同选项卡中会有一些面板相似。各个面板的使用信息在面板的说明中提供。要查看此信息，请单击面板名称右侧的问号按钮。
