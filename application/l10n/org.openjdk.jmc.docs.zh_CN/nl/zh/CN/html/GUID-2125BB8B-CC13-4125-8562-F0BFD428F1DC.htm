添加定制 JVM 连接

可以将连接添加到收藏 JVM，以便始终在 JVM 浏览器中显示它们。

要添加新 JVM 连接，请执行以下操作：

  1. 单击 JVM 浏览器顶部的创建新定制 JVM 连接按钮。  
    或者，打开文件菜单，选择连接，然后选择创建新连接，再单击下一步。      
  2. 在新建连接向导中，输入要连接的主机名和端口号，并指定连接名称。
  3. 如果希望测试连接并验证是否已输入正确的值，请单击测试连接。
  4. 必要时，指定用户名和口令进行验证，并选择在设置文件中存储身份证明，以使用主口令对其进行加密。有关详细信息，请参阅“在设置文件中存储身份证明”。
  5. 单击完成结束定制连接设置，或者单击下一步选择工具来连接到 JVM。

    此时该向导会测试连接。如果测试成功，将显示下一屏幕。

  6. （可选）如果在上一步单击了下一步，请在用于新连接的各列出操作（例如启动 JMX 控制台或启动飞行记录）中选择一个操作。
  7. 单击完成。
