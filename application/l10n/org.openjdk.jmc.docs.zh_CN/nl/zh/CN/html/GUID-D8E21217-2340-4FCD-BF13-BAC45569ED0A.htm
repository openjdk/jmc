使用“MBean 浏览器”选项卡

JDK Mission Control 使用托管 Bean (MBean) 为监视和管理 Java 应用程序性能提供统一且一致的界面。

MBean 是采用符合 JMX 规范的设计模式的托管对象。MBean 可以表示一个设备、一个应用程序或需要托管的任何资源。MBean 的管理界面由一组属性、操作和通知组成。

MBean 浏览器提供对所有已注册 MBean 的访问。MBean 在管理服务器中注册，后者可以通过与 Java Management Extensions (JMX) 兼容的客户机访问，例如 JDK Mission Control 中的 JMX 控制台。

MBean 浏览器选项卡分为以下面板：

  • MBean 树：所有已注册 MBean 的列表（按域列出）。可以使用筛选器字段按 MBean 名称筛选列表。要切换自动更新，请使用 MBean 树面板右上角的按钮。

    要创建和注册新 MBean，请单击 MBean 树面板右上角的绿色 "+" 图标。执行此操作会启动动态创建和注册新的 MBean 向导，提示您为新 MBean 输入对象名和类名。若要继续，对象名必须有效，并且类名必须是有效的 Java 类名。请注意，该向导不会验证类是否对 MBean 服务器可用；将只进行语法检查。要注销特定 MBean，请右键单击并从上下文菜单中选择删除。

  • MBean 功能：该面板包含以下子选项卡：

    • 属性：列出所选 MBean 的属性。

    • 操作：列出可从所选 MBean 调用的操作。

    • 通知：列出在运行时期间 MBean 所提示的通知。

    • 元数据：包含描述 MBean 的信息。
