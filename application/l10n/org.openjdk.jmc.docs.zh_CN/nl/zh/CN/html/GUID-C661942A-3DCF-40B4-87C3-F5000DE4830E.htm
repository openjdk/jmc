配置飞行记录器设置

可以使用 JDK Mission Control 首选项窗口配置飞行记录器插件的设置。

要修改飞行记录器设置，请执行以下操作：
  1. 打开“窗口”菜单，然后选择“首选项”。
  2. 在左侧面板中，展开 JDK Mission Control 并选择“飞行记录器”。
  3. 单击重置用户界面以便将飞行记录器插件的用户界面设置重置为默认状态。
  4. 如果不希望在服务器上保留已完成的记录，请选择从服务器中删除已完成的记录
  5. 如果要确认删除模板，请选择删除飞行记录模板之前要求确认。
  6. 选择在双击正在运行的记录时使用的转储默认时间范围。
    可以选择以下选项：

    • 没有默认值：显示要求您指定时间范围的窗口。

    • 整个记录：转储所有收集的数据。

    • 记录的最后部分：允许您在以下要转储的时间范围（最后部分）字段中指定要转储的记录的持续时间。在值后添加 s 可指定秒，添加 min 可指定分，添加 h 可指定小时，添加 d 可指定天，添加 w 可指定周。

  7. 单击应用以应用更改。
