使用“GC 配置”页

GC 配置页包含与垃圾收集相关的配置数据。

此页包含 GC 配置、堆配置和年轻代配置的面板。

使用“GC 配置”页：“GC 配置”面板

  • 年轻代垃圾收集器：年轻代所用的垃圾收集器。

  • 年老代垃圾收集器：年老代所用的垃圾收集器。

  • 并发 GC 线程：并发 GC 线程数。

  • 并行 GC 线程：并行 GC 线程数。

  • 并发显式 GC：指示 System.gc() 是否是并发的。

  • 已禁用显式 GC：指示 System.gc() 是否将导致垃圾收集。

  • 使用动态 GC 线程：是否使用了动态数量的 GC 线程。

  • GC 时间比率：GC 时间比率。

使用“GC 配置”页：“堆配置”面板

  • 初始堆大小：JVM 启动时的 Java 堆的大小。

  • 最小堆大小：为 Java 堆保留的最小内存量。

  • 最大堆大小：为 Java 堆保留的最大内存量。

  • 是否使用压缩 Oops：指示是否启用压缩普通对象指针。

  • 压缩 Oops 模式：指示使用的压缩普通对象指针的类型。

  • 堆地址大小：堆地址大小。
