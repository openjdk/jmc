MBean 浏览器首选项

本部分中的选项用于控制 MBean 在 JMX 控制台中的表示形式。

  • 在注销非系统 MBean 之前要求用户确认：在注销非系统 MBean 之前提示用户确认。此确认用于防止意外删除。

  • MBean 属性关键字顺序覆盖：确定 MBean 名称在 MBean 浏览器树中的排序顺序。如果更改此设置，则 MBean 浏览器中元数据选项卡上的 MBean 信息也将进行更新，即使创建顺序（注册 MBean 时指定属性的方式）保持不变也是如此。您可以指定应始终出现在 MBean 名称开头和结尾的属性，还可以指定这些属性应在 MBean 浏览器中出现的顺序。

  • MBean 后缀属性关键字顺序覆盖：该设置与上一设置相同，但影响 MBean 名称中的后缀。

  • 按字母顺序保留属性：指定是否按字母顺序显示保留的属性。

  • 属性关键字比较不区分大小写：指定比较属性关键字来确定顺序时是否将大小写考虑在内。

  • 显示压缩后的路径（省去属性关键字）：指定是在 MBean 名称中显示完整路径还是省去属性关键字。
