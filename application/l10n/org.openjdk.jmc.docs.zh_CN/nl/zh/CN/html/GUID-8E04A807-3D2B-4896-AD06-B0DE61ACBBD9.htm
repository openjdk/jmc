使用“VM 操作”页

VM 操作页包含 VM 操作的时间和持续时间统计信息。

此页以表、图形和子选项卡形式显示 VM 操作数据。

使用“VM 操作”页：选择和方面

使用页顶部的下拉菜单选择选项和方面（如果需要）。通过拖动图表区域（或选择表中的数据）并从上下文菜单中选择存储选择或存储选择并设置为焦点选择，可以将其他页中的数据保存到选择下拉菜单。

下面的控件提供其他行为：

  • 显示并发：显示对于所选方面并发的所有事件。

  • 交集：显示与原始事件有交集的所有事件，或者仅显示包含在同一时间范围内的事件。

  • 所有线程：显示所有线程中的事件，或者仅显示与原始事件相关的线程中的事件。

使用“VM 操作”页：表和图形

表中各列的标签如下：

  • VM 操作：VM 操作的种类。

  • 最长持续时间：此 VM 操作的最长持续时间。

  • 总持续时间：此 VM 操作的总持续时间。

  • StdDev(P)：持续时间标准差。

  • 计数：执行此 VM 操作的次数。

此页还包含另外三个子选项卡：时间表、持续时间或事件日志。

时间表

此子选项卡上的图形沿时间表绘制 VM 操作。

持续时间

此子选项卡上的图形按持续时间绘制 VM 操作。

事件日志

该子选项卡上的此表提供了以下信息：

  • VM 操作：VM 操作的种类。

  • 阻止：是否阻止调用 VM 操作的线程。

  • 安全点：VM 操作是否在安全点上发生。

  • 持续时间：VM 操作的持续时间

  • 调用方：请求操作的线程。

使用“VM 操作”页：配置规则

通过单击编辑配置图标，可以设置与此页关联的规则的配置属性。

可以使用以下选项：

已偏置锁定撤销

  • 已偏置锁定警告限制时间：触发警告所需的阻塞撤销已偏置锁定所用的总时间。

VM 操作峰值持续时间

  • 阻止 VM 操作持续时间警告限制：触发警告所需的阻止 VM 操作最短持续时间。
