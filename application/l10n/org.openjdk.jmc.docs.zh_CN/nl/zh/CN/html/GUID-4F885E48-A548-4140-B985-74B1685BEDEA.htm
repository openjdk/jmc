使用“TLAB 分配”页

TLAB 分配页包含有关线程本地分配缓冲区 (TLAB) 的信息。

此页以表和图形形式显示 TLAB 分配数据。

使用“TLAB 分配”页：选择和方面

使用页顶部的下拉菜单选择选项和方面（如果需要）。通过拖动图表区域（或选择表中的数据）并从上下文菜单中选择存储选择或存储选择并设置为焦点选择，可以将其他页中的数据保存到选择下拉菜单。

下面的控件提供其他行为：

  • 显示并发：显示对于所选方面并发的所有事件。

  • 交集：显示与原始事件有交集的所有事件，或者仅显示包含在同一时间范围内的事件。

  • 所有线程：显示所有线程中的事件，或者仅显示与原始事件相关的线程中的事件。

  • 时间范围：缩放到页上的相关位置的时间范围，或者清除时间范围缩放。

使用“TLAB 分配”页：表和图形

表中各列的标签如下：

  • 线程：线程名称。

  • 计数：此 TLAB 中的分配数。

  • 平均 TLAB 分配：TLAB 内部分配的估计平均大小。

  • 平均 TLAB 外部分配：TLAB 外部分配的估计平均大小。

  • 估计TLAB 分配：TLAB 中估计的分配大小。

  • TLAB 外部的总分配：TLAB 外部分配的总大小。

您可以选择以下任一项以便在图形上显示：

  • 估计TLAB 分配：TLAB 中估计的分配大小。

  • TLAB 外部的总分配：TLAB 外部分配的总大小。
