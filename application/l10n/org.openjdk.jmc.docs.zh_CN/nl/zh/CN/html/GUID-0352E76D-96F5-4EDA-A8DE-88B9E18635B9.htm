使用“异常错误和错误”页

异常错误和错误页包含有关由 Java 应用程序引发的异常错误和错误的信息。

此页以表的形式显示异常错误和错误信息。单击按类、按消息或按类和消息图标可以更改表中显示的数据的排列方式。

使用“异常错误和错误”页：选择和方面

使用页顶部的下拉菜单选择选项和方面（如果需要）。通过拖动图表区域（或选择表中的数据）并从上下文菜单中选择存储选择或存储选择并设置为焦点选择，可以将其他页中的数据保存到选择下拉菜单。

下面的控件提供其他行为：

  • 显示并发：显示对于所选方面并发的所有事件。

  • 交集：显示与原始事件有交集的所有事件，或者仅显示包含在同一时间范围内的事件。

  • 所有线程：显示所有线程中的事件，或者仅显示与原始事件相关的线程中的事件。

  • 时间范围：缩放到页上的相关位置的时间范围，或者清除时间范围缩放。

使用“异常错误和错误”页：表

表中各列的标签如下：

  • 类：异常错误或错误类的名称。

  • 消息：异常错误或错误提供的消息。

  • 计数：引发此异常错误或错误的次数。

可以通过下列子选项卡获取其他信息：

  • 时间表：包含绘制一段时间内的异常错误或错误数的图形。

  • 事件日志：显示表的事件日志信息，其中包括类、消息、线程和结束时间。

使用“异常错误和错误”页：配置规则

通过单击编辑配置图标，可以设置与此页关联的规则的配置属性。

可以使用以下选项：

引发的错误

  • 错误限制：触发警告所需的错误事件数。

  • 排除的错误类：描述要排除的错误类的正则表达式。

引发的异常错误

  • 异常错误限制：触发警告所需引发的异常错误数。
