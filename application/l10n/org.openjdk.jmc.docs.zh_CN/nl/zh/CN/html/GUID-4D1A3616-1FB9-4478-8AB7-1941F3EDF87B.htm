使用和定制 JVM 浏览器

JVM 浏览器包含可用的 JVM 连接列表。您可以展开列表并右键单击具有此类连接的节点，以显示可用的服务、操作和设置。JVM 连接支持拖放操作。

要定制 JVM 浏览器设置，请执行以下操作：
  1. 打开窗口菜单，然后选择首选项。
  2. 在左侧窗格中，展开 JDK Mission Control，然后选择 JVM 浏览器。
  3. 输入节点突出显示时间（以毫秒为单位），该时间用于定义具有新搜索到的 JVM 连接的节点在 JVM 浏览器树中突出显示的时间长短。
  4. 选择确认删除节点对从 JVM 浏览器树中删除 JVM 连接的提示进行确认。
  5. 如果需要此类警告，请选择未检测到本地 JVM 时发出警告。
  6. 单击应用以应用更改。
