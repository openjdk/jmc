通信设置

下列选项控制 JMX 控制台如何与 JVM 通信。

  • 默认更新间隔 [毫秒]：定义 JMX 控制台更新该属性所使用的时间间隔。

  • 保留的事件值：允许一个订阅（例如，计算机 CPU 负载等订阅属性）使用的最大值数。

  • 邮件服务器 (SMTP)：定义发送通知时使用的邮件服务器的地址。

  • 邮件服务器端口：定义指定的邮件服务器要使用的端口号。

  • 安全邮件服务器 (SSL)：指定通过 SMTP 进行的通信是否应使用 SSL 验证。

  • 邮件服务器用户：定义发送通知的电子邮件地址。

  • 邮件服务器口令：定义指定的电子邮件地址的口令。
