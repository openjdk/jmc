查看应用程序预警

每次触发带有应用程序预警操作的活动规则时，对应的预警将添加到触发预警窗口中。每次重新启动 JDK Mission Control 时将重置已触发应用程序预警的列表。

要打开触发预警窗口，请执行以下操作之一：

  • 打开窗口菜单并选择预警。

  • 将 JMX 控制台连接到任意 JVM，打开触发器选项卡，然后单击预警。

“触发预警”窗口列出自 JDK Mission Control 启动并连接到 JVM 以来，已经为 JVM 触发的所有应用程序预警。预警的列表包含以下列：

  • 日期：触发预警的日期。

  • 时间：触发预警的时间。

  • 规则：触发预警的规则的名称。

  • 源：触发了预警的 JVM 连接在 JVM 浏览器中显示的名称。对于远程 JVM 连接，此字段包含 IP 地址和端口号。

在列表中选择预警的时候，其详细信息显示在列表下方的字段中。根据所选操作事件显示不同的信息。

使用显示关于预警的对话框复选框可以切换是否希望只要触发了应用程序预警就打开“触发预警”窗口。

要清除触发的应用程序预警列表，请单击清除。或者，只要重新启动 JDK Mission Control 就会清除列表。
