转储默认记录

JVM 的默认记录类似于飞机上的黑盒。它在启动 JVM 时启动，并连续运行，该记录可用于分析崩溃、故障和其他事件。

要启用默认记录，请通过将 -XX:FlightRecorderOptions 选项设置为 defaultrecording=true 来启动 Java 应用程序。请不要忘记取消锁定商业功能和启用 JFR。

例如，要启用 Java 应用程序 MyApp 的默认记录，请使用以下命令：

java -XX:+UnlockCommercialFeatures -XX:+FlightRecorder -XX:FlightRecorderOptions=defaultrecording=true MyApp

要转储默认记录，请在 JVM 浏览器中的对应连接下右键单击飞行记录器节点，然后选择转储默认记录。此外，还可以右键单击名为 HotSpot 默认值的记录，并选择转储。此过程其余部分与转储任何正在运行的记录相同。请参阅：转储正在运行的飞行记录。
