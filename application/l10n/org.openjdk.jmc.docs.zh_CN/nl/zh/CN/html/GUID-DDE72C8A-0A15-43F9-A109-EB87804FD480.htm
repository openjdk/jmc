使用堆栈跟踪

堆栈跟踪窗口包含所记录事件的聚集堆栈跟踪。

使用此窗口可分析导致大量事件的堆栈跟踪中的调用。

使用堆栈跟踪：选择选项

以下按钮在“堆栈跟踪”窗口顶部的一行中显示：

  • 选择帧组：

  • 上一个帧组：

  • 下一个帧组：

  • 组跟踪自上一个方法帧：

  • 组跟踪自线程根：

  • 视图菜单：显示各种分组、格式设置和布局选项。

  • 最小化：最小化此窗口。

  • 最大化：最大化此窗口。

使用堆栈跟踪：查看结果

默认情况下，堆栈跟踪在交互树视图中显示其结果。通过展开或隐藏以三角形图标开头的行，可以浏览堆栈跟踪的各个部分。

该页顶部的视图菜单图标提供下列选项：

  • 组堆栈跟踪自

    • 线程根：

  • 帧的判别方式

    • 优化类型：

    • 字节码索引：

    • 行号：

    • 方法：

    • 类：

    • 程序包：

  • 布局选项

    • 显示为树：

    • 减少树深度：

  • 方法格式设置选项
    • 返回值

      • 隐藏

      • 类名

      • 类和程序包名称

    • 类

      • 隐藏

      • 类名

      • 类和程序包名称

    • 参数

      • 隐藏

      • 类名

      • 类和程序包名称
