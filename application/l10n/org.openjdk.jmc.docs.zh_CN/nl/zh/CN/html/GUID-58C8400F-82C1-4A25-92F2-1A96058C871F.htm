使用大纲

大纲是一种导航工具，让您可以通过单击鼠标来访问各个页。

可以通过从上下文菜单中选择相应选项来添加或删除页。以下部分介绍默认情况下显示的页。本文档的后面部分以单独的主题形式对下面列出的每一页进行了更详尽的介绍。

使用大纲：导航树

  • Java 应用程序

    • 内存：包含有关一段时间内的内存分配、使用和垃圾收集的信息。

    • 锁定实例：包含有关 Java 应用程序中的锁定实例事件的信息。

    • 文件 I/O：包含有关在文件系统中执行的读取/写入活动的信息。

    • 套接字 I/O：包含有关在套接字中执行的读取/写入活动的信息。

    • 异常错误和错误：包含有关由 Java 应用程序引发的异常错误和错误的信息。

  • JVM 内部

    • 垃圾收集：包含与 GC 活动相关的信息。

    • GC 配置：包含与垃圾收集相关的配置数据。

    • 编译：包含记录期间所执行的方法编译的相关信息。

      • 代码高速缓存：包含有关 JVM 代码高速缓存的信息。

    • 类加载：包含记录期间加载和卸载的类的相关信息。

    • VM 操作：包含 VM 操作的时间和持续时间数据。

  • 环境

    • 进程：包含有关记录期间运行的进程的信息。

    • 环境变量：包含执行记录的系统的环境变量详细信息。

    • 系统属性：包含执行记录的系统的系统属性信息。

    • 记录：包含有关记录及其事件设置的信息。

  • 事件浏览器
