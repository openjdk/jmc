屏幕阅读器

JDK Mission Control 支持多种屏幕阅读器，这些阅读器可以将屏幕信息翻译成口语，以便为有视觉障碍的用户提供帮助。

配置选项当前适用于由 Freedom Scientific, Inc. 生产的 JAWS 屏幕阅读器。有关配置此产品的信息，请访问 Freedom Scientific 屏幕阅读器网站，网址为 http://www.freedomscientific.com

注：

如果您正在使用 JAWS，请注意切换选项卡不会按预期方式工作。有关解决方法，请参阅“在选项卡之间切换”。
