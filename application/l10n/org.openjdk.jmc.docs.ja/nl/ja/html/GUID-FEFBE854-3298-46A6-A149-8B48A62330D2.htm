「GC構成」ページの使用

「GC構成」ページには、ガベージ・コレクションに関連する構成データが含まれます。

このページには、「GC構成」、「ヒープ構成」および「若い世代の構成」のパネルが含まれます。

「GC構成」ページの使用: 「GC構成」パネル

  • 若いガベージ・コレクタ: 若い世代に使用されるガベージ・コレクタ。

  • 古いガベージ・コレクタ: 古い世代に使用されるガベージ・コレクタ。

  • 並行GCスレッド: 並行GCスレッドの数。

  • パラレルGCスレッド: パラレルGCスレッドの数。

  • 並行明示的GC: System.gc()が並行かどうかを示します。

  • 明示的GC無効: System.gc()によってガベージ・コレクションが発生するかどうかを示します。

  • 動的GCスレッドを使用します: GCスレッドの動的な数が使用されるかどうかを示します。

  • GC時間比: GC時間の比率。

「GC構成」ページの使用: 「ヒープ構成」パネル

  • 初期ヒープ・サイズ: JVM起動時のJavaヒープのサイズ。

  • 最小ヒープ・サイズ: Javaヒープのために予約されている最小メモリー容量。

  • 最大ヒープ・サイズ: Javaヒープのために予約されている最大メモリー容量。

  • 圧縮Oopsが使用される場合: 圧縮された通常のオブジェクト・ポインタを有効にするかどうかを示します。

  • 圧縮Oopsモード: 使用中の圧縮された通常のオブジェクト・ポインタの種類を示します。

  • ヒープ・アドレス・サイズ: ヒープ・アドレス・サイズ。

  • オブジェクト整列:

「GC構成」ページの使用: 「若い世代の構成」パネル

  • 若い世代の最小サイズ: 若い世代の最小サイズ。

  • 若い世代の最大サイズ: 若い世代の最大サイズ。

  • 新規比率:

  • 初期殿堂入りしきい値:

  • 最大殿堂入りしきい値:

  • 使用されているTLAB: スレッド・ローカル割当てバッファ(TLAB)が使用中であるかどうかを示します。

  • 最小TLABサイズ: 最小のTLABサイズ。

  • TLABリフィル浪費上限:
