タブおよびタブ・グループ

ビューとエディタは、タブ・グループに結合されたタブとして表示されます。タブとタブ・グループをJMCクライアント・ウィンドウにドラッグして、最適なレイアウトを作成できます。ビュー・タブおよびタブ・グループは、メイン・ウィンドウから切り離すこともできます。

タグ・グループをJMCクライアント・ウィンドウのサイズに最大化したり、他のタブ・グループの場所を空けるために最小化したりできます。

  • タブ・グループを最大化するには、右上隅にある「最大化」ボタンをクリックします。タブ・グループはJMCクライアント・ウィンドウとほぼ同じサイズに拡張され、「最大化」ボタンが復元ボタンに変わります。他のタブ・グループは自動的に最小化されます。

  • タブ・グループを最小化するには、右上隅にある「最小化」ボタンをクリックします。タブ・グループが縮小され、「最小化」ボタンが復元ボタンに変わります。他のタブ・グループには、残りのJMCクライアント・ウィンドウが比例的に表示されます。

  • タブを元の状態に戻すには、復元ボタンをクリックします。

論理的に関連する情報を分離するために、エディタ・タブがサブタブ(またはサブタブ・グループ)に分割され、特定のデータにドリル・ダウンできます。一部のサブタブには、データを制御するボタンがサブタブ名の右に表示されます。
