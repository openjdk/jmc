スクリーン・リーダー

JDK Mission Controlでは、視覚障害のあるお客様を支援するために、画面ベースの情報を音声に変換するスクリーン・リーダーをサポートしています。

現在、Freedom Scientific, Inc.のJAWSスクリーン・リーダーの構成オプションが使用可能です。この製品の構成については、次のFreedom Scientificスクリーン・リーダーのWebサイトを参照してください: http://www.freedomscientific.com

注:

JAWSを使用する場合、タブの切替えが予期したとおりに動作しないことに注意してください。回避策は、「タブ間の切替え」を参照してください。
