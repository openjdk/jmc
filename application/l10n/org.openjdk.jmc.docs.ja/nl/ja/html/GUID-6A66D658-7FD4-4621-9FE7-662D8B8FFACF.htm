「クラスのロード」ページの使用

「クラスのロード」ページには、記録時のクラスのロードおよびアンロードに関する情報が含まれます。

このページでは、クラス・ロード・データが表とグラフの両方に表示されます。

「クラスのロード」ページの使用: 選択内容と側面

必要に応じて、ページの一番上にあるドロップダウン・メニューを使用して、選択内容と側面を指定します。他のページのデータを選択内容ドロップダウン・メニューに保存するには、チャートの領域をドラッグして(または表からデータを選択して)、コンテキスト・メニューから「選択内容の格納」または「格納して、フォーカスされた選択内容として設定」を選択します。

次のコントロールで追加の動作を指定します。

  • 同時の表示: 選択した側面に並行するイベントをすべて表示します。

  • 交差: 元のイベントと交差するすべてのイベントまたは同じ期間に含まれるイベントのみを表示します。

  • すべてのスレッド: すべてのスレッドのイベントまたは元のイベントに関連するスレッド内のイベントのみ表示します。

「クラスのロード」ページの使用: 表とグラフ

グラフに表示する次の任意の項目を選択できます。

  • クラスのロード: 間隔中にロードされたクラス数。

  • クラスのアンロード: 間隔中にアンロードされたクラス数。

  • ロードされたクラス数: JVMの起動以降ロードされたクラス数。

  • ロードされていないクラス数: JVMの起動以降ロードされていないクラス数。

表の列は、次のようにラベル付けされています。

  • 定義クラス・ローダー: 定義クラス・ローダー。

  • ロードされたクラス: このクラス・ローダーがロードしたクラス数。

  • アンロードされたクラス: アンロードされたクラス数。

このページには、「クラスのロード」および「クラスのアンロード」サブタブも含まれます。

「クラスのロード」タブ

  • ロードされたクラス: ロードされたクラスの名前。

  • 期間: クラスのロードの時間。

  • 定義クラス・ローダー: 定義クラス・ローダーの名前。

  • 開始クラス・ローダー: 開始クラス・ローダーの名前。

  • スレッド: クラスがロードされたスレッド。

  • 終了時間: クラス・ロードのイベントの終了時間。

「クラスのアンロード」タブ

  • ロードされていないクラス: ロードされていないクラスの名前。

  • 定義クラス・ローダー: 定義クラス・ローダーの名前。

  • 終了時間: クラス・アンロードのイベントの終了時間。

「クラスのロード」ページの使用: ルールの構成

「構成の編集」アイコンをクリックすると、このページに関連付けられたルールの構成属性を設定できます。

次のオプションを使用できます。

クラス・ロード・プレッシャ

  • クラスロード期間制限: 警告をトリガーする必要のある最短クラスロード期間。

  • クラスロード率制限: クラスロードで費やされた時間と記録の合計期間の間の最小比率。
