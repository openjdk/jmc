ボタンのテキスト・ラベルの表示

JDK Mission Controlには、テキスト・ラベルなしのグラフィック・ボタンが多数あります。スクリーン・リーダーで読取り可能なテキスト・ラベルを表示する手順:

  1. 「ウィンドウ」メニューを開き、「プリファレンス」を選択します。
  2. 左のペインで「JDK Mission Control」を選択します。
  3. 「アクセシビリティ・オプション」で、「ボタンのテキスト・ラベルを表示(新規に接続を開き、記録した後に有効になります)」を選択します。
  4. 「適用」をクリックします。
実行中のプラグインを再起動すると、テキスト・ラベルがボタンに表示されます。
