通知の表示

通知は、サーバー上のイベントに関する情報を提供します。一部のMBeanでは、このような情報が提供されず、MBeanブラウザの「通知」サブタブを使用して、通知をサブスクライブする必要があります(通知がある場合)。

通知をサブスクライブするには、MBeanを選択して、「サブスクライブ」を選択します(「情報」パネルに表示されている場合)。下の「ログ」パネルに、各通知が個別のエントリとして表に追加されます(表には、通知の日時で構成された名前が表示されます)。エントリを展開し、通知の詳細を表示します。
