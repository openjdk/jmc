診断コマンドのプリファレンス

このセクションのオプションは、診断コマンドの実行時に表示される警告を制御します。

特定の影響のある診断コマンドに対して警告を表示しない場合、該当するチェック・ボックスを選択します。

  • 影響が中規模である診断コマンドを警告なしで実行します。

  • 影響が大規模である診断コマンドを警告なしで実行します。

  • 影響が不明な診断コマンドを警告なしで実行します。
