JMXデータ永続性の設定

「JMXデータ永続性の設定」パネルは、JMXコンソールの「概要」タブの上部に縮小されています。永続的に保存可能な属性のリストが含まれます。

リストに属性を追加するには、「JMXデータ永続性の設定」パネルの右上隅にあるプラス記号付きの「属性の追加」ボタンをクリックし、「追加する属性の選択」ウィンドウで属性を選択します。詳細は、属性の操作を参照してください。

リストから属性を削除するには、属性を右クリックして、「削除」を選択します。

リストの属性のデータ永続性を有効にするには、「履歴データの設定」パネルの右上隅の「JMXデータ永続性のアクティブ化」ボタンをクリックします。
