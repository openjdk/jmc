アクセシビリティの既知の問題と回避策

この項には、JDK Mission Controlのアクセシビリティ機能の知識を高めるための追加手順が含まれます。

1行のみのツリー表でのナビゲート

1行のみが含まれるツリー表コンポーネント内でナビゲートする場合は、[Space]キーまたは[Shift]キーを押し、[↑]および[↓]キーを使用して行にナビゲートします。

タブ間の切替え

タブ・コンポーネントにアクセスする際、JAWSは「ページを切り替えるには、[Ctrl]キーを押しながら[Tab]キーを押します」と誤ってユーザーに伝える場合があります。タブを切り替える正しい方法は、左または右の矢印キーの使用です。

スクリーン・リーダーでの表データの読上げ

表データを画面読上げソフトウェアでより効率的に読み上げるには、表データをテキスト・エディタにコピー・アンド・ペーストし、そこから読み上げます。データをコピーするには、対象の行を選択し、右クリックして「コピー」を選択します。次に、データをテキスト・エディタに貼り付けます。
