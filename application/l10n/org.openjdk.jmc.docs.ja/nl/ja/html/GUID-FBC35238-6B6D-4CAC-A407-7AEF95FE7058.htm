オンライン・ヘルプ・テキストのサイズ変更

視覚障害のあるお客様にとって、テキスト・サイズを大きくしないかぎり、スタンドアロン(RCP)バージョンのJDK Mission Controlのオンライン・ヘルプ・ドキュメントを読むのは難しい場合があります。Eclipseのヘルプ・ビューアにおける制限のために、現在はテキストのサイズを変更できません。フォント・サイズを変更するには、外部ブラウザでヘルプを表示します。

  1. 「ウィンドウ」メニューを開き、「プリファレンス」を選択します。
  2. 左のペインで「ヘルプ」を選択します。
  3. オープン・モードで、ヘルプ・コンテンツを外部ブラウザに開くことを選択します。
  4. 「適用」をクリックします。
JDK Mission Controlではオペレーティング・システムで指定されたデフォルトのWebブラウザが使用され、JDK Mission Control内から別のWebブラウザを指定することはできません。オペレーティング・システムでのブラウザの指定方法は、オペレーティング・システムによって異なります。
