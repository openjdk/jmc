パネル

エディタ・タブは、論理的に関連する情報をグラフ、表、リストなどの形式で表示するパネルで構成されます。ボタン、テキスト・フィールド、ドロップダウン・リストなど、パネル内の様々なコントロールを使用して、一部のパラメータを変更できます。

一部のパネルを縮小して、タブ内の他のパネル用の場所を空けることができます(パネル名の横の黒い三角形の表示/非表示ボタンを使用します)。パネル名の右には、コンポーネントの追加と削除、データのリフレッシュ、モードの切替え、設定の変更、パネルに関するオンライン・ヘルプの表示などを行うボタンがあります。

ほとんどのパネルは固有ですが、一部は様々なタブについて類似しています。各パネルの使用に関する情報は、パネルの説明にあります。この情報を表示するには、パネル名の右にある疑問符ボタンをクリックします。
