ダイヤルの追加または編集

「ダッシュボード」パネルにダイヤルを追加できます。

ダイヤルを追加するには、パネルの右上隅の「属性の追加」ボタンをクリックして、「追加する属性の選択」ウィンドウの属性を選択します。

ダイヤルのプロパティを編集する手順:

  1. ダイヤルを右クリックし、「表示名の編集」を選択してダイヤルの名前を設定します。
  2. ダイヤルを右クリックし、「プロパティ」を選択してダイヤルのグラデーションおよびウォーターマークの値を設定します。グラデーションの開始および終了値を指定して、ダイヤルの色を調整します。「色」ボタンをクリックし、開始色と終了色およびウォーターマークの色を選択して、色を調整することもできます。
  3. ウォーターマークを表示するかどうかを選択します。
  4. 「OK」をクリックします。

「ダッシュボード」パネル内でダイヤルを移動するには、ダイヤルを右クリックして、「左に移動」または「右に移動」を選択します。

ダイヤルを削除するには、ダイヤルを右クリックして、「削除」を選択します。

ウォーターマークを属性の現在値にリセットするには、ダイヤルを右クリックして、「統計のクリア」を選択します。これは、アプリケーションを長時間モニタリングしており、属性の最大値が異なるほどパフォーマンスが変更された場合に便利です。
