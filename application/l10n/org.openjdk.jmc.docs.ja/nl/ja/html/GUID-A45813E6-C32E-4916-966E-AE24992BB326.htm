「JVM内部」ページの使用

「JVM内部」ページには、記録時に実行されていたJVMに関する情報が含まれます。

このページには、「JVM情報」および「JVMフラグ」のパネルが含まれます。

「JVM内部」ページの使用: 「JVM情報」パネル

  • JVM起動時間: JVMが起動された日付と時刻。

  • JVM名: JVMの名前。

  • JVMバージョン: JVMのフル・バージョン。

  • JVM引数: JVMの起動に使用されたjavaコマンドのオプションを含みます。

  • JVMアプリケーション引数: アプリケーションの実行に使用されたクラス・ファイルまたはJARファイルと、main()メソッドに渡された引数を含みます。

「JVM内部」ページの使用: VMフラグ

「JVMフラグ」表には、次の列があります。

  • 名前: フラグの名前。

  • 値: フラグの値。

  • 元: フラグの元(Default、Ergonomic、Command lineまたはManagementのいずれか)。Defaultオプションがデフォルトで設定されます。Command Lineオプションは、コマンド行に指定されます。Ergonomicオプションは、環境に基づいてJVM自体により人間工学的に設定されます。
