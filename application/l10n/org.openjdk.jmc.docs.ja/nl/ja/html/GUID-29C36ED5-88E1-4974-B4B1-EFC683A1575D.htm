ダッシュボード

「ダッシュボード」パネルは、単一の属性の現在値を表示するダイヤル(パフォーマンス・メーター)で構成されます。

ダイヤルは、値とポインタが表示された色付きのスケールです。ダイヤルの上部にはタイトルが表示されます。各ダイヤルの中央には、現在の単位が表示されます。ダイヤルの下部には、属性の現在値(ポインタの現在の位置に対応)および最大値(ポインタの最も高い観測位置であるウォーターマークに対応)が表示されます。タイトル、単位、およびスケールとウォーターマークの両方の色は調整可能です。

デフォルトでは、次のダイヤルが表示されます。

  • 使用されているJavaヒープ・メモリー

  • JVM CPU使用率

  • ライブ・セット + 断片化
