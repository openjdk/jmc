ローカルで検出されたJVM

JVMブラウザは、コンピュータで実行中のローカルに検出されたJVMをリスト表示します。

これらのJVMには、JMXRMIが設定されていない場合でも、JDK Mission Controlプラグインを使用して接続できます。ローカルで検出されたJVMに接続できない場合、JMXRMIをJVMのバージョンがデフォルトでサポートしていないか、JMXRMIの手動による有効化が必要である可能性があります(たとえば、64ビット・バージョンのJDK Mission Controlは32ビットJVMのJMXRMIを有効化できません)。
