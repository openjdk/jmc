「プロセス」ページの使用

「プロセス」ページには、記録時に実行されていたプロセスに関する情報が含まれます。

このページでは、プロセス・データが表とグラフの両方に表示されます。

「プロセス」ページの使用: 選択内容と側面

必要に応じて、ページの一番上にあるドロップダウン・メニューを使用して、選択内容と側面を指定します。他のページのデータを選択内容ドロップダウン・メニューに保存するには、チャートの領域をドラッグして(または表からデータを選択して)、コンテキスト・メニューから「選択内容の格納」または「格納して、フォーカスされた選択内容として設定」を選択します。

次のコントロールで追加の動作を指定します。

  • 同時の表示: 選択した側面に並行するイベントをすべて表示します。

  • 交差: 元のイベントと交差するすべてのイベントまたは同じ期間に含まれるイベントのみを表示します。

  • すべてのスレッド: すべてのスレッドのイベントまたは元のイベントに関連するスレッド内のイベントのみ表示します。

「プロセス」ページの使用: 表とグラフ

グラフには、CPU使用率と同時プロセスに関する情報がプロットされ、時間に沿って表示されます。

表の列は、次のようにラベル付けされています。

  • プロセス識別子: プロセス識別子(PID)。

  • コマンド行: このプロセスを開始したコマンド。

  • 最初のサンプル: このプロセスがサンプリングされた最初の時間。

  • 最終サンプル: このプロセスがサンプリングされた最後の時間。

「プロセス」ページの使用: ルールの構成

「構成の編集」アイコンをクリックすると、このページに関連付けられたルールの構成属性を設定できます。

次のオプションを使用できます。

競合しているCPU使用率

  • 競合しているCPU使用率制限: 情報通知のトリガーに必要とされる、他のプロセスによって使用されるCPU量。

  • 競合しているCPU使用率警告制限: 警告のトリガーに必要とされる、他のプロセスによって使用されるCPU量。

競合しているプロセス

  • 競合しているプロセス制限: 情報通知のトリガーに必要とされる、同時プロセスの数。
