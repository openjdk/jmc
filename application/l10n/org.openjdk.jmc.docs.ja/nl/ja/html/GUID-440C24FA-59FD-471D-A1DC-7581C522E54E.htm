「概要」タブの使用

JMXコンソールを起動すると、デフォルトで「概要」タブが開きます。ダイヤルとグラフのセットと、選択した属性のリアルタイムの使用量の統計が表示されます。

デフォルトでは、「概要」タブに、閉じた状態の「JMXデータ永続性の設定」パネル、「ダッシュボード」パネル、および「プロセッサ」と「メモリー」の名前の2つのグラフが表示されます。デフォルトのレイアウトでは、CPUおよびメモリーの使用量の概要が表示されます。ダイヤルとグラフのセットを変更して、最も重要な属性値の現在の状態の概要を表示できます。

「概要」タブをデフォルトの状態にリセットするには、タブの右上隅の「デフォルト・コントロールにリセット」をクリックします。
