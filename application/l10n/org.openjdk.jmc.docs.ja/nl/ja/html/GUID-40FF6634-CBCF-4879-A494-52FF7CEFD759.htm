記録の表示

フライト・レコーダ・プラグインでは、JFR記録ファイルを表示できます。

JFRランタイム・エンジンは、記録されたデータをjfr拡張子のファイルにダンプします。これらのJFR記録は、JDK Mission Control (JMC)クライアントで表示するためのバイナリ・ファイルです。

注:

Java SE Development Kit (JDK) 7 Update 4以降のバージョンがJFR記録の表示に必要です。

時間制限のある記録が終了するか、実行中の記録のダンプが作成される場合、記録ファイルがJMCクライアントで自動的に開きます。ダブルクリックするか、「ファイル」メニューを開いて「ファイルを開く」を選択し、ファイル・システム・エクスプローラで必要な記録ファイルを特定して、記録ファイルを開くこともできます。

このトピックでは、JMCクライアントの記録を表示するための一般的な手順について説明します。様々なタブのデータの編成方法の詳細は、「フライト・レコーダのタブ・リファレンス」を参照してください。
