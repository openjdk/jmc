Java Discovery Protocol経由で検出されたJVM

Java Discovery Protocol (JDP)では、JVMブラウザでネットワーク上のJVMインスタンスをリスト表示できます。

JDPのプリファレンスを設定する手順:
  1. 「ウィンドウ」メニューを開き、「プリファレンス」を選択します。
  2. 左ペインで、「JDK Mission Control」を展開して、「JVMブラウザ」、「JDP」の順にクリックします。
  3. 参加するマルチキャスト・グループのアドレスを入力します。
  4. リスニングするポートを入力します。
  5. JVMハート・ビートのタイムアウトを入力します。
  6. 「適用」をクリックして変更を適用します。

注:

変更内容を有効にするには、JDK Mission Controlを再起動する必要があります。
