操作の実行

操作とは、管理対象のオブジェクト内のJMXサーバーでコードを実行するメカニズムです。

操作は、メソッドのようにパラメータを使用して結果を戻すことができます。JDK Mission Controlは、これらの操作の実行を、パラメータの受渡しおよび結果の表示とともに、サポートします。

MBeanブラウザの「操作」サブタブには、選択したMBeanに関連付けられた操作が表示され、これらの操作を実行できます。戻り型は、リストの操作の名前の横に表示されます。右側の表には、操作の実行に必要なパラメータのリストが表示され、各パラメータの値を設定できます。

操作を実行する手順:

  1. 実行する操作を選択します。
  2. アプリケーションで必要なすべてのパラメータの値を指定します。
  3. 表の下の「実行」をクリックします。操作の結果は別のパネルに表示され、実行された各操作がそのタブに表示されます。
