通信設定

次のオプションは、JMXコンソールがJVMと通信する方法を制御します。

  • デフォルトの更新間隔[ミリ秒]: JMXコンソールが属性を更新するために使用する時間間隔を定義します。

  • 保存イベント値: 1つのサブスクリプションでの使用を許可する値の最大量(たとえば、マシンCPU負荷などのサブスクライブされた属性)。

  • メール・サーバー(SMTP): 通知の送信時に使用されるメール・サーバーのアドレスを定義します。

  • メール・サーバー・ポート: 指定されたメール・サーバーに使用するポート番号を定義します。

  • セキュアなメール・サーバー(SSL): SMTP経由の通信でSSL認証を使用するかどうかを指定します。

  • メール・サーバー・ユーザー: 通知の送信元の電子メール・アドレスを定義します。

  • メール・サーバー・パスワード: 指定された電子メール・アドレスのパスワードを定義します。
