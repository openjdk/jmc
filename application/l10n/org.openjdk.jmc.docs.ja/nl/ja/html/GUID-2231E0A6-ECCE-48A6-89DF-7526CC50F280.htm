JDK Mission Controlのアクセシビリティ・モード

JDK Mission Controlには、傾向を明らかにして、パフォーマンス・メトリックの最小値と最大値を特定できるようにする、データの有効なグラフィカル表示を提供するパフォーマンス・データのダイヤルとグラフが表示されます。

ただし、ダイヤルとグラフでは、スクリーン・リーダーで読取り可能な方法では情報を伝達しません。この問題を修正するために、JDK Mission Controlでは、ダイヤルとグラフのデータを表形式で表示する方法が提供されており、視覚に障害のあるユーザーでもアクセスしやすくなっています。アクセシビリティ・モードを切り替えるには、ダイヤルまたはグラフの関連するパネルの右上隅の「アクセシビリティ・モード」ボタンをクリックします。

JDK Mission Controlのすべてのパネルでアクセシビリティ・モードを有効にする手順:

  1. 「ウィンドウ」メニューを開き、「プリファレンス」を選択します。

  2. 左のペインで「JDK Mission Control」を選択します。

  3. 「アクセシビリティ・オプション」で、「アクセシビリティ・モードの使用」を選択します。

  4. 「適用」をクリックします。
