グラフ

デフォルトでは、「概要」タブに、「プロセッサ」と「メモリー」の2つのグラフが表示されます。「プロセッサ」グラフは、コンピュータおよびJVMによるCPU使用状況が表示されます。「メモリー」グラフには、JVMによるシステム・メモリーおよびJavaヒープの使用状況が表示されます。

グラフを「概要」タブに追加するには、タブの右上隅の「チャートの追加」ボタンをクリックし、次に必要な属性をグラフに追加して、必要に応じてその外観を変更します。

グラフ・パネルを削除するには、グラフ・パネルの右上隅の「閉じる」ボタンをクリックします。

グラフの属性の追加および削除の詳細は、属性の操作を参照してください。グラフの使用の詳細は、グラフの使用を参照してください。
