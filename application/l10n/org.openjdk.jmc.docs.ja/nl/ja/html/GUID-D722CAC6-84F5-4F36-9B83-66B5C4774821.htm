7 試験的プラグイン

JDK Mission Controlの試験的プラグインは、JDK Mission Controlを様々な方法で拡張したり、このような拡張の開発を支援するために設計されたプラグインのセットです。

試験的プラグインは、次のいずれかです。

  • フィードバックに応じて、後でメインの配布に組み込むことがでるテクノロジ・プレビュー。

  • Java Flight RecorderのWebLogic Serverプラグインなど、Java SEに直接には関連しない機能でJDK Mission Controlを拡張するプラグイン。

注:

これらのプラグインは、現在サポートされておらず、評価の目的でのみ提供されています。試験的プラグインは、JDK Mission Control製品配布に含まれていません。

JDK Mission Controlクライアントを使用して、試験的な更新サイトから試験的プラグインをダウンロードおよびインストールできます。

プラグインをインストールする手順:

  1. 「ヘルプ」メニューを開き、新規ソフトウェアのインストールを選択します。

  2. インストールするプラグインを選択し、インストール・ウィザードに従います。

注:

インストールされたプラグインを使用するには、JDK Mission Controlを再起動します。
