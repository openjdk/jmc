「記録」ページの使用

「記録」ページには、記録とそのイベント設定に関する情報が含まれます。

このページでは、その情報が表とグラフに表示されます。

「記録」ページの使用: 表とグラフ

「並行記録」表

  • 記録名: 記録の名前。

  • ID: 記録のID。

  • 記録の開始: 記録の開始時間。

  • 記録時間: 記録の継続時間。

「イベント設定」グラフ

  • 設定の対象: この設定の対象を示します。

  • 名前の設定: 設定名。

  • 設定値: 設定値。

「記録」ページの使用: ルールの構成

「構成の編集」アイコンをクリックすると、このページに関連付けられたルールの構成属性を設定できます。

次のオプションを使用できます。

消失フライト・レコーダ・バッファ

  • バッファ消失制限: 警告のトリガーに必要とされる消失バッファ・イベントの数。
