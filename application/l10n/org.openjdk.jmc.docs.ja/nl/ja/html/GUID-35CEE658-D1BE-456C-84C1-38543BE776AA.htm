フライト記録テンプレートの管理

フライト記録テンプレートは、記録されるイベントおよびイベント詳細のセットを定義します(一部のイベントでは、スタック・トレースをイベントにアタッチし、期間しきい値およびリクエスト期間を指定できます)。

フライト記録テンプレート・マネージャを開く手順:
  1. JVMブラウザでJVMを右クリックして、「フライト記録の開始」を選択します
  2. フライト記録の開始ウィザードの「テンプレート・マネージャ」をクリックします。
    JFRには、2つの構成済のサーバー側の記録テンプレートが含まれています。

    • 連続: 1%未満のオーバーヘッドの本番環境で連続して使用する構成済のテンプレート。このテンプレートは、デフォルトの記録に使用されます

    • プロファイリング: 約2%のオーバーヘッドのプロファイリングの構成済のテンプレート。

構成済のテンプレートは変更できませんが、要件に応じて複製して複製を変更できます。テンプレートを複製するには、リストで選択して、「複製」をクリックします。これにより、同じ名前の複製テンプレートが作成され、名前にカッコで囲まれた数値が追加されます。
