使用“跳转到源代码”

要从 JDK Mission Control Client 跳转到源代码，请执行以下操作：


注：

以下过程是通用的。有关支持此功能的插件列表，请参阅支持“跳转到源代码”的 JDK Mission Control 插件。

  1. 在列出类或方法的表、树或其他 GUI 组件上，右键单击要查看其源代码的类或方法。

    此时将显示一个上下文菜单 (图：上下文菜单上的“跳转到源代码”命令)。

    上下文菜单上的“跳转到源代码”命令


    “上下文菜单上‘跳转到源代码’命令”的说明

  2. 选择打开方法 (或打开类型，如果要从类调用中跳转)。

    关联的源代码将显示在新窗口中。
