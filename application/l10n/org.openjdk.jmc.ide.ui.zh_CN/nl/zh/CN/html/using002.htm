Eclipse IDE 版和 RCP 版 JDK Mission Control 之间的差异

一般来说，Eclipse 版本的 JDK Mission Control Client 与 RCP 版本作用相同。Eclipse 版本中的任何组件都与 RCP 上提供的类似组件提供相同的功能和用户界面。要在安装后切换到 JDK Mission Control，请转至 Mission Control 透视图。

Eclipse 版本和 RCP 版本之间最大的差异在于“跳转到源代码”功能。“跳转到源代码”(如跳转到应用程序源代码中所述) 是通过将监视和概要分析工具与开发环境进行紧密耦合来实现的。“跳转到源代码”用于从 Mission Control 中显示的任何类、方法或框架打开和查看相应源，条件是源位于工作区源路径中。
