选择透视图

透视图定义 Eclipse 窗口内的一组视图及其相对位置；换句话说，它是以图形方式显示 Eclipse 中不同类型的信息的模板。例如，Java 透视图组合了在编辑 Java 源文件时常用的视图，而“调试”透视图包含了在调试 Java 程序时将使用的视图。

Eclipse 的 JDK Mission Control 插件提供一个名为 Mission Control 的预定义透视图。此透视图显示 JDK Mission Control 用户界面，以便当您在 Eclipse 中开发应用程序时可以使用构成 JDK Mission Control 的工具对这些应用程序进行概要分析。

要打开 JDK Mission Control 透视图，请执行以下操作：

  1. 在 Eclipse 窗口的右上角，单击打开透视图图标。

    此时将显示“打开透视图”上下文菜单。

  2. 选择其他。

    此时将显示“打开透视图”对话框。

  3. 选择 Mission Control，然后单击确定。

    Eclipse 窗口将重新配置以显示 Mission Control 透视图。

要从 JDK Mission Control 更改透视图，请执行以下操作：

  1. 如果从未打开该透视图，则单击打开透视图图标，然后选择要打开的透视图。

    如果上下文菜单中未显示该透视图名称，则选择“其他...”以打开“打开透视图”对话框，然后从中选择透视图。

  2. 如果以前打开过该透视图，则在“打开透视图”对话框旁边，标准 Mission Control 透视图的右上角将显示一个该透视图的按钮。单击要打开的透视图的按钮。

要重新打开 Mission Control 透视图，请执行以下操作：

如果已为此项目打开 Mission Control 透视图，则 Eclipse 右上角的“打开透视图”按钮旁边将显示一个“Mission Control”按钮。单击该按钮。
