关于 Eclipse 插件

Eclipse 插件版的 JDK Mission Control 提供 JDK Mission Control 的应用程序概要分析和监视工具集与 Eclipse 开发平台的无缝集成。通过将 JDK Mission Control Client 与 Eclipse 集成，可以将 Eclipse 的功能与构成 JDK Mission Control 的工具相结合。JDK Mission Control - Eclipse 插件提供有关使用通过将 JDK Mission Control Client 与 Eclipse 进行集成启用的特殊功能的说明。

在 Eclipse IDE 内运行 JDK Mission Control Client 时，可以访问将 JDK Mission Control Client 作为独立的富客户端平台 (RCP) 应用程序运行时在工具中不可用的 IDE 功能。这些功能中最重要的是通过直接从 JDK Mission Control Client 中打开源代码来查看正在运行的应用程序中源代码的功能，此功能称为“跳转到源代码”。

将 JDK Mission Control Client 与 Eclipse IDE 集成的另一优点包括允许您在开发阶段对应用程序进行概要分析和监视，就像在生产阶段一样。这使您可以在将应用程序实际部署到生产环境中之前识别可能的运行时问题。
